

## 一个树莓派，v3 Model B

[see RaspberryPi](./docs/PI.jpg)

## 安装Raspbian 系统

[官网文档安装](https://www.raspberrypi.org/downloads/raspbian/)

## 购买一些电子元器件（万能的淘宝和华强北）

- 面包板
- 面包线
- LED灯
- 全彩LED灯
- 数码管（各种二位，三位）
- 电阻若干
- 杜邦线（公对公，母对母，公对母）
- USB转TTL线

## 准备开发环境

```
sudo apt-get install python-dev
sudo apt-get install python-rpi.gpio

# apt-cache search gpio # many gpio libs

```
